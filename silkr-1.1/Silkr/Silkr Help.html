Disable Overlay

Disables or enables the virtual cursor's overlay. Disabling the overlay will increase performance.

All Screens (Experimental)

Attempts to extend the virtual cursor's overlay to all screens if applicable. This is very performance heavy.

Manual Overlay Override

Enables the overlay override text inputs and overrides the location and size of the overlay. Overriding the overlay to occupy a smaller size will increase performance.

Overlay Size

Sets the overlay's width in terms of x pixels and height in terms of y pixels.

Overlay Position

Sets the overlay's position in terms of x and y in pixels.

Hotkeys

.

Hotkeys can be set for the following functions:

• Smooth On/Off - Toggles smoothing on or off.
• Overlay On/ - Toggles the overlay on or off.
• Toggle Display - Cycles through the available monitors to display the overlay on.
• Tablet Mode - Manually toggles Tablet Mode when Disable Auto Detection is off.
• Strength Up - Increases the smoothing strength by 10 in increments of 10.
• Strength Down - Decreases the smoothing strength by 10 in increments of 10.

Tablet cursor jumps between itself and virtual cursor

You will need to disable Windows/Microsoft Ink in your tablet driver's settings. Most if not all Wacom tablets should have this option as a checkbox when setting up program profiles.

How to get this to work on Photoshop

1. Disable Windows/Microsoft Ink for Photoshop in your tablet driver/profiler
2. Navigate to your Photoshop settings folder. It should be located at:
C:\Users\YOURUSERNAME\AppData\Roaming\ Adobe\Adobe Photoshop CC\Adobe Photoshop CC Settings\
3. Locate the settings file: PSUserConfig.txt
4. If it does not exist, create it.
5. Add the following line inside that file:
UseSystemStylus 0

This will enable Photoshop's legacy handling for tablets.

Under Settings -> Cursor check Disable Catch Up.

Tablet pen is drawing elsewhere

Under Settings -> Tablet check Tablet Offset Override. You can offset the virtual cursor and real cursor with x and y in terms of pixels.
